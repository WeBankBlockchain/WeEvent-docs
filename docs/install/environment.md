## 系统要求

### 配置说明

| 配置   | 最低配置                 | 推荐配置         |
| ---- | -------------------- | ------------ |
| CPU  | 2核 1.5GHz            | 4核 2.4GHz    |
| 内存   | 2GB                  | 4GB          |
| 带宽   | 1M                   | 5M           |
| 操作系统 | CentOS、Ubuntu、RedHat | CentOS       |
| Java | Java(TM) 1.8         | Java(TM) 1.8 |

### 第三方软件

| 软件名     | 推荐版本 | 是否可选 |
| ---------- | -------- | -------- |
| Nginx      | 1.14.2   | 必选     |
| FISCO-BCOS | 2.0.0    | 必选     |
| Mysql      | 5.6.x    | 可选     |
| Redis      | 5.x.x    | 可选     |
| Zookeeper  | 3.5.x    | 可选     |
| WeBase     | 1.0      | 可选     |
