=========================================
安装部署
=========================================
本章节主要介绍WeEvent的安装部署。WeEvent部署安装分为快速安装和详细安装。

快速安装：通过快速安装，可以搭建一个WeEvent基础环境，可以方便用户体验核心功能。


详细安装：在生产环境中，建议用户参考详细安装，部署全部功能，保证系统高效和稳定。
