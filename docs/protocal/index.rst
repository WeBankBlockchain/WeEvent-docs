接入说明
========================================= 
WeEvent服务用Java Spring Boot框架实现。

+ 为Java业务接入提供了Java SDK客户端，SDK不依赖Spring框架。
+ 为支持其他语言，提供了RESTful、JsonRPC风格的协议接入。
+ 为方便系统集成，提供了STOMP协议的支持。传输协议支持SockJS和WebSocket两种。
+ 为方便物联网IOT设备的使用，提供了MQTT协议的支持。
